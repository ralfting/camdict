# Camdict

CLI of Cambridge Dictionary

![Camdict demo](https://i.imgur.com/WTkqGIf.gif)

## Description

A simple command line to use Cambridge Dictionary in the terminal

## Getting Started

### Installing

> npm i camdict -g

### How use it

```
camdict <word> // Exp.: $ camdict house
camdict "<phrase>" // Exp.: $ camdict "wrap up"
camdict --help
```
